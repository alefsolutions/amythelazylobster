# amythelazylobster
It's Amy, not as great as OpenClaw, just lazy and minimalist.

It's not there yet, but it is getting there.

I tried OpenClaw, I loved it. I'm still quite overwhelmed by it.

The whole idea of an autonomous agent sounds great, but it's a concept I'm still getting used to.

For now, getting things done over my phone with Amy my lazy assistant makes me a little bit comfortable. I don't need it to do things autonomously, just when I ask it to and I'm content with that. E.g., "Schedule an event for tomorrow at 4pm", "Send me the most recent report for client A", all while I'm away from my computer. This is already a big jump in my productivity.

## How it Works?
For now, it's very simple.

User Prompt > Telegram > LLM > tool > Execute Action/Tool/Script > LLM > Telegram > User.

LLM reasons (OpenAI for now), picks up the tool to do the job (e.g. run a script, etc.), gets results, reasons, and sends response to user.

No Soul.md, Heartbeat.md, and other extra fluff...just Me.md.
